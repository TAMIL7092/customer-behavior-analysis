# Customer Behavior Analysis

## Description
Analyzing customer shopping behavior using data analysis and visualization techniques to identify patterns, trends, and insights for business decision-making.

## Project Files
- Jupyter Notebook: Data analysis and EDA
- Power BI Dashboard: Customer behavior visualization
- CSV Dataset: Customer shopping data
- Business Problem Document: Business understanding

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Outcome
This project demonstrates practical data analysis skills and business-oriented thinking.
